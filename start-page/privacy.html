The extension does not collect or transmit any private information. It just displays your apps and bookmarks in the way that you arranged them. Simply disable or uninstall this extension if you want to get rid of it.

You're always free to check the source code of this extension by using the Chrome inspection tools. Also contact me at klovadis@gmail.com if you notice anything fishy.

Data collection

  • This extension does not collect any data. Only your settings are being saved.
  • This extension does not send any data away from your computer.
  • If you uninstall this extension, all data will be deleted forever.

This extension can access

  • Your bookmarks: Obviously, this extension needs to access your bookmarks in order to show them. It does not manipulate them however.
  • Manage themes, extensions, and apps: In order to display your installed apps, this extension needs access to them. Since there is no other interface in Chrome, it has to be able to uninstall an app for you if you choose to.
  • chrome://favicon/: To get the shiny icons for your bookmarks, this extension needs access to the favicons stored internally.
  • Your browsing history: When asking for the bookmarks permission above, this warning also pops up. However, this extension does not access any history data.

Requests to webservers

  • This extension loads images from www.gvm-it.eu.
  • This extension loads a list of predefined themes from www.gvm-it.eu.

No private data is being sent. By loading theme information from that website, I can create more themes without forcing users to update the extension all the time and keep its filesize small. Just so you know.
