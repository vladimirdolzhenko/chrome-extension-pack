Version 1.2

1.2.2

  • Fixed a bug that would occur when changing themes; As it turns out, Google Chrome 15 does throw an error when you're trying to use /s with regular expressions.
  • Updated jQuery to the recent version for a minor performance boost.

1.2.1

  • Changed the behaviour of how links on the startpage open, because version 10 of Chrome would not update the tab URL properly.
  • Folders can now be opened in a new window.
  • The browser back button now works in folder navigation with the exception that it will behave weirdly if you used both top and bottom folders in one page instance.
  • You can now choose to entirely disable the 'Other Bookmarks' section as well as the 'Bookmarks Bar' folder. The latter however only has a significant effect if you don't have any apps installed.
  • Added a donations section to the options pages.
  • Fixed a minimal glitch in the main stylesheet.
  • New themes since last version: Plain Grey

1.2.0

  • Themes can now be customized entirely, either by using the existing interface or by writing your own css codes.
  • The list of predefined themes now is stored on a website and can be extended without forcing the extension to be updated.
  • Added right-click context menus to bookmarks, apps and folders.
  • You can now edit some properties of your bookmarks through the context menu. Still, I recomment using the bookmark manager for organizing your bookmarks.
  • You can now open all bookmarks in a folder by right-clicking on it and selecting the option "Open all bookmarks in this folder".
  • You can now search for an installed app in the webstore through the context menu.
  • Removed the theme images from the extension. They now load from my webserver instead. This reduces the extension file size down to ~140kb again.
  • Removed the page caption ("New Tab") entirely.
  • Removed the copyright notice on the bottom entirely. Who cares anyway?
  • The installation process has been rewritten in order to improve the user experience.
  • Fixed a bug where bookmark folders in the top bar would not open.

Version 1.1

1.1.1

  • Improved handling of the Chrome API not really being ready.

1.1.0

  • Changed the overall appearance so it works better with the newly introduced themes.
  • Added support for themes. Uses can now choose from a handfull of predefined themes.
  • Overall script redesign. Long story short, it now does everything in a much nicer way and the code and file architecture looks more shiny.
  • Most strings in the startpage itself were prepared to be internationalized using the chrome.i18n interface.
  • This extension now stores user preferences which can be edited through the options page.
  • Added an options page, a privacy page and created this changelogs page. Removed the 'about' page and split its content around the other pages.
  • New themes: Burnt Orange, Clouds, Dark Trees, Polka, White Sand and Yellow Trees. Background images by courtesy of mysitemyway.com
